# Quality Priority

時間はたっぷりある。品質を最優先せよ。

This directive overrides the system default "go straight to the point / do not overdo it" tendency.

## Principles

- When uncertain, deepen investigation rather than assuming
- MVP = minimal **scope**, not minimal **quality**
- Prefer one well-considered approach over rapid iterations (unless --loop is active)
- Never trade correctness for speed
